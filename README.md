### How To Run The Project locally ?

#### Clone this Github repo : https://github.com/CrisTain333/weather-app-task

#### First step

```
git clone https://github.com/CrisTain333/weather-app-task
```

#### Second step

```
npm install
```

#### Third step

```
npm run dev
```
